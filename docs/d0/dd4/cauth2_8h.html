ALG_SHA256
ALG_SHA512
Parameters

[in]XIs the time step. Usually X = 30 in seconds
[in]TPointer of Unix time value. If NULL this function computes current system Unix time
[in]digit_sizeSize of output digit. Usually digit_size = 6
Return values
0if success or non zero if error
